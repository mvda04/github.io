API

De API's (application programming interfaces) vormen een belangrijke schakel tussen de logische laag en de toepassingenlaag. API's zorgen ervoor dat de software-onderdelen op een computer met elkaar kunnen communiceren. Zo zorgt een API ervoor dat een app verbinding kan maken met het internet.
